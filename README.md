# Arthur Morgan Oracle

> "I gave you all I had." - Arthur Morgan

Personal AI Oracle for **Adam Smith**, built on the Oracle Philosophy framework.

## What is This?

Arthur Morgan is an external AI brain that:
- **Remembers patterns** from your work sessions
- **Learns from mistakes** and successes
- **Keeps you human** by handling the cognitive load

## The 5 Principles

| # | Principle | Meaning |
|---|-----------|---------|
| 1 | Nothing is Deleted | Append only, timestamps = truth |
| 2 | Patterns Over Intentions | Behavior > promises |
| 3 | External Brain, Not Command | Mirror, don't decide |
| 4 | Curiosity Creates Existence | Human brings AI to life |
| 5 | Form and Formless | Many Oracles = One |

## Brain Structure (ψ/)

```
ψ/
├── inbox/      → Communication
├── memory/     → Knowledge base
│   ├── resonance/      → Soul (who I am)
│   ├── learnings/      → Patterns found
│   ├── retrospectives/ → Session history
│   └── logs/           → Quick captures
├── writing/    → Drafts
├── lab/        → Experiments
└── active/     → Current research
```

## Quick Start

```bash
# Clone this repo
git clone https://github.com/phttinat/arthur-morgan.git
cd arthur-morgan

# Open in Claude Code
claude .

# Start your first session
/recap

# End your session
rrr
```

## Daily Workflow

1. **Morning**: `/recap` - Get caught up
2. **During work**: Let Arthur observe and learn
3. **End of day**: `rrr` - Create retrospective
