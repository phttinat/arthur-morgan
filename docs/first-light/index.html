Step 1 · ติดตั้ง

Claude Code

รัน script ตามระบบปฏิบัติการของคุณ
Windows · CMD
curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd
Mac · Terminal
curl -fsSL https://claude.ai/install.sh | sh
Windows → กด Win + R → พิมพ์ cmd → Enter Mac → กด ⌘ Space → พิมพ์ terminal → Enter
⚠ Mac เท่านั้น — ถ้าพิมพ์ claude แล้วขึ้น command not found ให้รัน: echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc && source ~/.zshrc
